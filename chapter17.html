Chapter 17: HTML5 Layout

This chapter covers other ways to organize a layout for your page, but by using other tags in your HTML file other than div.

New Elements

Header and Footer

Header and Footer elements can be used for the top and bottom of every page, or for an individual section or article (more on these in a bit.) For example, on a page with several blog posts, each individual post can be thought of as a separate section. Theheader element can therefore be used to contain the title and date of each individual post, and the footer might contain links to share the article on social networking sites.

Article

Article is useful for any part of the page that could stand alone. Like an individual article, blog entry, forum post, or comment.

Aside

Asides have a couple of uses. When used in conjunction with an article, an aside could be used as a pullquote or a glossary. It should contain information related to the article. When used outside of an article, it should contain information related to the entire page. Links to other sections of the site, list of recent posts, search box, recent tweets, etc.

Section

Sections group related content together. Typically each section would have its own heading. An example would be latest news, top products, newsletter signup would all be an individual section. Each section could have multiple articles inside. The section element should NOT be used for the entire page. If you want one containing element for the entire page, you're better off using the div element.

Figure and Figcaption

back in Chapter 5, we talked about the Figure element. It's used to contain any content that is referenced from the main flow of an article (not just images). You should generally use figure for videos, images, graphs, diagrams, code samples, and text that supports the main body of the article. Figcaption is placed inside the figure element, and used to give a text description of what is inside the figure.

Back to div

After going through the uses for all the above elements (and doing a little further reading,) if you don't think you should be using any of them then you're better off just using a div tag.

Block link

You can turn an entire block level element into a link by containing it into an a element. Simply add <a href="link.html"> to the top, and </a> to the bottom.
