Chapter 9: Flash, Video, and Audio

Although flash does not have as large of a presence as it used to, I feel like I should still know how to embed it into a page.

Embedding Flash

Adding flash can be easily done with the embed element, which is what I will show today. Later on, Javascript can be used to embed files which seems like the better way to go.
I opened the source page to find a direct link to this swf file for testing purposes.


The code used here:
<embed src="linktoswffile.swf">

HTML5 Video

Embedding HTML5 video is fairly straightforward. One thing to be aware of, there are 2 formats suported by different browsers.

  • H264
    • Internet Explorer
    • Safari
  • WebM
    • Android
    • Chrome
    • Firefox
    • Opera

Multiple Video Sources

When using multiple video sources, you need to take some precaution. It's best to include the video type, and codecs used. Below is an example of how to do this. Take notice of when " is used, as well as ' Also take note that .mp4 comes first, because iPads are picky.
<video poster="images/puppy.jpg" width="400"
height="320" preload controls loop>
<source src="video/puppy.mp4" type='video/
mp4;codecs="avc1.42E01E, mp4a.40.2"' />
<source src="video/puppy.webm" type='video/
webm;codecs="vp8, vorbis"' />
<p>A video of a puppy playing in the snow</p>
</video>


Audio

We can add audio using flash mp3 players, or a default browser player. If you want to use flash, the best method would be uploading your file to a website like soundcloud and using their embedding code. But if you want to use your own code, you can use <audio>
