Here we have a wealth of information that I have not used yet. I imagine a lot of it I will not need to use for a long time, but there's no harm in learning it now (and having this page to reference later.)


Form Structure

<form action="http://www.example.com/subscribe.php" method="get" />
Form controls
</form>
Every form requires an action. The action page is the URL that will receive the information in the form once it's submitted.
The method will be either get(default value if undefined) or post. In general, get will be used for:

  • short forms like search boxes
  • when just retrieving data from a server

Post method will send values in what are known as HTTP Headers. Should use post for:
  • user uploadsing a file
  • longer form
  • sensitive data like passwords
  • adds or deletes information from a database
