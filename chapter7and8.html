Here we have a wealth of information that I have not used yet. I imagine a lot of it I will not need to use for a long time, but there's no harm in learning it now (and having this page to reference later.)


Form Structure

<form action="http://www.example.com/subscribe.php" method="get" />
Form controls
</form>
Every form requires an action. The action page is the URL that will receive the information in the form once it's submitted.
The method will be either get(default value if undefined) or post. In general, get will be used for:

  • short forms like search boxes
  • when just retrieving data from a server

Post method will send values in what are known as HTTP Headers. Should use post for:
  • user uploadsing a file
  • longer form
  • sensitive data like passwords
  • adds or deletes information from a database

While the password input type hides the characters from view, it is not a secure way to hide information. If you need a secure method, use SSL for things like credit card information.

Text Area

What did you think of this gig?


You can use <textarea>to create an input that will include multiple lines. The code above looks like the following:
<textarea name="comments" cols="20" rows="4">Enter your comments...</textarea>
You don't need to use the rows and columns tags, instead you should use CSS to control the height/width of the text box.

Radio Button

Please select your favorite genre:
Rock Pop Jazz

Input type Radio is used for the above code. Can only select one option, cannot deselect. Looks like the following:
<input type="radio" name="genre" value="rock" checked="checked" /> Rock
You will need to create one line of these for each option, changing only the value each time. The text to appear next to each button will be whatever you type right after the "greater than" bracket.

Checkbox

Please select your favorite music service(s):
iTunes Last.fm Spotify

The Checkbox input type uses the same code as Radio, except as you probably could guess, inputput="checkbox".

Dropdown List Box

What device do you listen to music on?

The Drowdown List Box should be used instead of radio buttons if you have a long list of options. The code looks like the following:
<form action="http://www.example.com/profile.php">
<p>What device do you listen to music on?</p>
<select name="devices">
<option value="ipod">iPod</option>
<option value="radio">Radio</option>
<option value="computer">Computer</option>
</select>
</form>

Multiple Select Box

Do you play any of the following instruments? (You can select more than one option by holding down control on a PC or command key on a Mac while selecting different options.)

You can use a Multiple Selection Boxwhen you have a longer list of options. Eventually I will learn better ways to choose from multiple options, but for now this will do as a learning tool. The code looks like the following: <form action="http://www.example.com/profile.php">
<p>Do you play any of the following instruments? (You can select multiple options by holding down the control key on a PC or the command key on a MAC.)<p>
<select name=instruments" size="3" multiple="multiple">
<option value="guitar" sellected="selected">Guitar</option>
<option value="drums">Drums</option>
<option value="heyboard">Keyboard</option>
<option value="bass">Bass</option>
</select>
</form>

File Input Box

Upload your song in MP3 format:


The File Input Boxcan be used for uploading any type of file to a server/website. When using this code, when entering the <form> tag, the method must be equal to post.
<input type="file" name="whatever you would like" />
Under this code you will need to add another line which allows you to submit the file. You can include a name="whatever" attribute if you like, but it is not necessary.
<input type="submit" value="Whatever text you would like to appear" />

Image Button

In order to create an Image Button all you have to is use the following code:
<input type="image" src="image_source.jpg" />
You can also add height/width attributes.
