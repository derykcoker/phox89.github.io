Chapter 3: Lists and Chapter 4: Links

These chapters were full of information I already knew, so I will only be adding in a small amount of new information.


You can use <dl>, <dt>, and <dd> to make a list of definition. Below is an example

<dl> to section off the html code.

<dt>Vocabulary 1</dt>
<dd> Here is the definition for vocabulary word 1.<dd>
<dt>Vocabulary 2</dt>
<dd> Here is the definition for vocabulary word 2.<dd>

</dl> to finish off the code section.


When creating a link, you can add in _blank to open the link in a new tab. For example <a href="index.html" target="_blank">New tab</a> opens a link in a new tab.

Lastly, you can jump to certain parts of a page by # symbol.
For example, I have given the <h1> element at the top of the page that says HTML Notes the id top. If I type <a href="#top"#62 Back to top</a> then you'll see a link like this that sends you back to the top of the page.
