Chapter 10

Chapter 5: Images and Chapter 6: Tables

Images are fairly straight forward. There is some code to help organize them better, however the book states you are better off organizing them with CSS so currently I will not be worrying about trying to give them better placement.
To add an image, you will want to type the following:
<img src="url to image" alt="alternate text to show if the image is unable to load" title="title of image that shows when mouse hovered over" />

You can also use <figure> surrounding the code with the image. Inside this code you can include <figcaption> to include a caption under the image.

<table>, <tr> (table row), and <td> (table data) are used to create tables. Use table to start a table, tr to start a new row, and td to enter data into a table cell.


To title rows/columns you can use table heading. The code looks like <th scope="col"> or <th scope="row">


To help organize your code, you can use <thead> in the section of code to section off the head. As well as <tbody> for the main area of cells, and <tfoot> for the last row(s).


Lastly, you can use <td colspan="2"> or <td rowspan="2"> to have your data span across multiple rows or columns. Below are a couple tables showing the use of all these tags.
