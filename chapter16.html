Chapter 16: Images

This chapter focuses on how to size and align images affectively in your HTML page.

Size and Alignment

To control the size of an image, you use the height and width properties like you would with other boxes. A good idea is to create classes for image sizes you would like to use (like small, medium, large) then define them in CSS with img.small, img.medium, img.large. When added images through html, just define the image with whichever class you would like to use.
Aligning images is done in a similar way. In HTML code the image will be given two classes: align-left or align-right, and then the size. In CSS you create a rule for img.align-left that includes float:left. You then create another for align-right. So in HTML the code would be:
<img src="imgsrc.jpg" class="align-left medium' alt="whatever">
Then in CSS you would add
img.align-left {
float: left;
margin: 10px;}
Adding a margin in is a great idea to keep everything separated.
By default, images are inline elements. If you would like to center one, you would have to convert it to a block level element by using display: block; in your CSS file (best to give the image a class such as align-center.) After that has been done, there are 2 ways to center the image:
Use text-align: center; on the containing element, or
On the image itself, set the left and right margins to auto.

Background Images

You can use background-image: url("image.jpg"); to set the background image for a box. If you set it to be the body element, it will be the background for the entire page. Background images are usually the last thing to load, so larger images are usually not recommended.
background-repeat: and background-attachment: are pretty self explanatory. Repeat can have 4 values: repeat (repeats on x and y axes,) repeat-x(repeats on x axis,) repeat-y(repeats on y axis,) and no-repeat. Attachment can have 2 values: fixed (background image stays in one spot) or scroll (image moves up and down as user scrolls on the page.)
background-position: can be used to position the image. You define it with 2 values: the first being the horizontal location (left, center, right) and the second the vertical (top, center, bottom.) You can also specify pixels, or a percentage. 0% 0% would be the top left. 50% 50% would center the image.

Background Shorthand

If you would like to combine these into one declaration, you can use them all with the property background: You must use them in the following order (although you can skip one if you would like):

  • background-color:
  • background-image:
  • background-repeat:
  • background-attachment:
  • background-position:
So all put into one rule would look like:
background: #ffffff url("image.jpg") repeat scroll left top;
