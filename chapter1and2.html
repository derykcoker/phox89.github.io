Chapter 2: Text

You can use <strong> to type bold characters


You can use <em> to type italics

You can use <sup> and <sub> to type 2nd and CO2

Have you ever used <blockquote> to add in a fake quote? If you need to indent text, you should use CSS!


If you hover the mouse over NASA, because I used the <abbr title="actual name"></abbr> tag you will be able to see what NASA stands for.

Also don't forget to use <cite> when citing a text. You can also use <strike> if you need to strike out text.
